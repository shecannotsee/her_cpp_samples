#### 描述

该项目展现了一个项目，如何通过cmake提供的功能，来在构建项目时构建第三方库的例子。前提是在指定目录下已经存在了第三方库的源代码。

#### 如何运行？

```bash
# 需要获取第三方库的源代码
$ pwd
.../cmake_demo
$ mkdir third_party
$ cd third_party
# 获取源代码
$ ../script/get_third_party_src.sh
$ cd ..
# 现在可以想其他cmake项目一样使用该项目了
$ mkdir build
$ cd build 
$ cmake ..
$ make -j2
......
```
